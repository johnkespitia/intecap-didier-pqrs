# Sistema de Control PQRS

## Descripción
Aplicación web para la gestión y seguimiento de Peticiones, Quejas, Reclamos y Sugerencias (PQRS).

## Características
- Registro de usuarios
- Creación de PQRS
- Seguimiento en tiempo real
- Panel administrativo
- Notificaciones por email
- Reportes y estadísticas

## Tecnologías
- React.js
- Node.js
- MongoDB
- Express
- Material-UI

## Instalación
```bash
git clone [repositorio]
cd pqrs-didier
npm install
npm start
```

## Configuración
1. Crear archivo `.env`
2. Configurar variables de entorno
3. Configurar base de datos

## Uso
1. Registrar usuario
2. Iniciar sesión
3. Crear PQRS
4. Dar seguimiento

## API Endpoints
- POST /api/pqrs
- GET /api/pqrs
- PUT /api/pqrs/:id
- DELETE /api/pqrs/:id
